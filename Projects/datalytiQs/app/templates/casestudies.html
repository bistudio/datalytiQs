Gender Pay Gap

A concise summary of the April 2023 median gender pay gap using ONS data, illustrated through a unique visual approach in 9 key job categories.

Approach

Using python and pandas, the data was cleaned and prepared for use in D3.js, which in turn was used to create a visualisation of the data and then rendered using python and flask.
